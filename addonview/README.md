# Information
This site generates and displays addon modules for [Meteor Client](https://meteorclient.com/)

# Showcase
Follow **[this link](https://zgoly.github.io/addonview/generate?user=Zgoly&repo=Meteorist&path=src/main/java/zgoly/meteorist/modules)** to see how it works

# How to use
For example, let's check this link

<pre>
zgoly.github.io/addonview/generate?<ins>user=Zgoly</ins>&<ins>repo=Meteorist</ins>&<ins>path=src/main/java/zgoly/meteorist/modules</ins>
</pre>

You can see 3 parameters:
1. `user` - your github username
2. `repo` - your github repository name
3. `path` - path to modules folder

- Using `user` you make it clear which profile used.
- `repo` - makes it clear which repository will be used to display modules.
- `path` - makes it clear in which folder files with modules are located.
