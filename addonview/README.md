# Information
This site generates and displays Meteor Client addon modules

# Showcase
Follow **[this link](https://zgoly.github.io/addonview/generate?user=Zgoly&repo=Meteorist&link=src/main/java/zgoly/meteorist/modules)** to see how it works

# How to use
You have 3 parameters:
1. user - your github username
2. repo - your github repository name
3. link - link to modules folder

You can make you own link using example below

## Example
`https://zgoly.github.io/addonview/generate?user=Zgoly&repo=Meteorist&link=src/main/java/zgoly/meteorist/modules`

This url has everything you need for normal display of modules
